# CMSC320-Semester-Project
Uses "Football Transfer Value Intelligence 2026" Dataset

## Setup Instructions

### 1. Clone the Repository

```
git clone <your-repo-link>
cd CMSC320-Semester-Project
```

---

### 2. Create and Activate Environment

```
conda create -n cmsc320 python=3.10
conda activate cmsc320
```

---

### 3. Install Dependencies

```
python -m pip install -r requirements.txt
```

---

### 4. Download the Dataset

Run:

```
python download.py
```

This will download the dataset into a local cache directory on your machine.

---

### 5. Place Dataset in Project Folder

After downloading, navigate to path

Copy the folder and paste it into the project
